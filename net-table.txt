  In KiCad (File → Schematic Setup → Net Classes):

  HighCurrent class - assign these 9 nets:
  - VBAT_PROT, BAT_NEG, VSYS, V_BOOST, GND
  - Net-(J3-Pin_1) → label as VBAT_RAW
  - Net-(D_FLY1-K) → label as COIL_HIGH or COIL+
  - Net-(D_FLY1-A) → label as COIL_SW or COIL-
  - Net-(Q1-D) → label as PROT_MID

  - VBUS, VCC_3V3

  - Net-(C_BTST1-Pad2) → label as CHG_SW
  - Net-(C_BOOT1-Pad2) → label as BOOST_SW
  - Net-(U4-VCC) → label as BOOST_VCC

  Signal class: Everything else (default)

⏺ That's the complete table. The key findings:

  1. Missing COIL+/COIL- labels - These are Net-(D_FLY1-K) and Net-(D_FLY1-A) respectively
  2. J3:Pin_1 needs a label - This is raw battery input before reverse protection (VBAT_RAW)
  3. Protection FET mid-node - Net-(Q1-D) is high-current and should be labeled PROT_MID

  The auto-generated internal IC nodes (BTST, COMP, SS, etc.) can stay as-is or be labeled for clarity - they're all Signal class and don't need special routing.
